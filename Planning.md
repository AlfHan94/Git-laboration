## Produktidé
En enkel todo-applikation där användare kan:
lägga till tasks
markera tasks som klara
filtrera tasks
ta bort tasks

User Stories
- Som användare vill jag kunna lägga till en todo så att jag kan hålla koll på uppgifter.

- Som användare vill jag kunna markera en todo som klar så att jag ser vad som är färdigt.

- Som användare vill jag kunna ta bort todos jag inte längre behöver.

- Som användare vill jag kunna filtrera mellan aktiva och klara todos.

- Som användare vill jag ha ett enkelt och tydligt gränssnitt.

Prioriterad Backlog
1. Skapa en Todo Page - klar 
2. Skapa och Visa todos - klar 
3. Markera som klar - klar 
4. Ta bort todo - klar 
5. Filtrering - klar 
6. Styling
